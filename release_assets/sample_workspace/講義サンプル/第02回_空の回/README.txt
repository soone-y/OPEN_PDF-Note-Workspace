これは、PDF 注釈の保存と読み込みを確認するサンプルセッションです。

- `注釈付きサンプル.pdf`: 元の PDF です。
- `注釈付きサンプル.clrop`: PDF に重ねて表示する注釈データです。

アプリで `注釈付きサンプル.pdf` を開くと、同じフォルダにある同名の `.clrop` が自動的に読み込まれます。PDF と `.clrop` は 1 組として一緒に扱ってください。`.clrop` をノートとして開いたり、手作業で編集したりしないでください。

このサンプルには、1ページ目の説明用テキスト、矢印、文字に沿ったマーカー、波線が入っています。注釈入りの PDF を別ファイルとして作る場合は、メニューの「出力」から「簡単: 注釈を統合してPDF出力...」を使います。元の PDF は上書きされません。

この PDF はプロジェクト内の合成テスト PDF を配布用に複製したものです。外部から取得した資料や個人情報は含めていません。配布前に全 5 ページの表示内容を確認済みです。

This is a sample session for checking PDF annotation saving and loading.

- `注釈付きサンプル.pdf`: the original PDF.
- `注釈付きサンプル.clrop`: annotation data displayed over the PDF.

When you open `注釈付きサンプル.pdf`, the same-named `.clrop` file in this folder is loaded automatically. Keep the PDF and `.clrop` together. Do not open or manually edit `.clrop` as a note.
