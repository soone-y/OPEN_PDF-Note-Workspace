このセッションには、現在の基本操作を案内する PDF と、次の 5 種類のノートが入っています。

- 使い方_基本操作.pdf: ワークスペースを開く、PDF に注釈を付ける、ノートを作る、保存する、注釈入り PDF を別ファイルへ書き出す、という現在の基本操作を案内します。注釈の練習用 PDF としても使えます。
- ノート_基本操作.clro: 本ソフトの標準ノート。Markdown と独自の補助記法の例です。
- ノート_数式.md: Markdown と数式の例です。
- ノート_TeX数式.tex: `.tex` 拡張子で数式を使う例です。
- ノート_プレーンテキスト.txt: 書式を解釈しない一般的なテキストの例です。
- ノート_表データ.csv: 表計算としては処理せず、プレーンテキストとして開く CSV の例です。

推奨するローカル確認手順:
1. `使い方_基本操作.pdf` を開きます。
2. 小さな注釈を追加します。
3. 拡張子ごとのサンプルノートを開いて、扱いの違いを確認します。
4. 元の PDF ファイル自体が上書きされていないことを確認します。

This session contains a PDF guide to the current basic workflow and five note examples.

Suggested local workflow:
1. Open one of the sample PDFs.
2. Add a small annotation.
3. Edit the sample note.
4. Confirm that the original PDF file itself is not overwritten.
