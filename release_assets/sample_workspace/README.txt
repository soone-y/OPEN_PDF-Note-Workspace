PDF Note Workspace サンプルワークスペース

このフォルダは、ポータブル配布物向けの安全な開始テンプレートです。
サンプル構成から始めたい場合は、このフォルダを複製して名前を `workspace` に変えて使えます。

内容:
- `workspace.json`: サンプル用の保守的なローカル専用設定です。
- `講義サンプル/第01回_基本操作/`: 基本操作を案内する PDF (`使い方_基本操作.pdf`) と `.clro`、`.md`、`.tex`、`.txt`、`.csv` のノート例です。
- `講義サンプル/第02回_最初期構想/`: 2025 年 12 月の最初期の画面案 2 件と、同時期の統合画面構成 PDF (`PDF学習ワークスペース統合画面構成および基本仕様書.pdf`) ・注釈データです。現行仕様ではなく、開発の出発点を示す参考資料です。
- `講義サンプル/第03回_Office変換/`: LibreOffice runtime で変換した Office-to-PDF の検証結果です。PowerPoint 由来 1 件と Word 由来 1 件の PDF を確認できます。

すべての PDF・Office 文書・画面案は、プロジェクト内で作成した合成サンプルです。外部から取得した資料や個人情報は含めていません。
配布アセットの PDF は確定配布サンプルであり、今後の変換品質向上に伴って積極的には差し替えません。

PDF Note Workspace sample workspace

This folder is a safe starter template for a portable release.
You can copy this folder and rename the copy to "workspace" if you want to start with the sample structure.

Contents:
- `workspace.json`: conservative local-only settings for the sample.
- `講義サンプル/第01回_基本操作/`: basic usage guide PDF (`使い方_基本操作.pdf`) and `.clro`, `.md`, `.tex`, `.txt`, and `.csv` note examples.
- `講義サンプル/第02回_最初期構想/`: two early UI mockups from December 2025, plus the contemporary integrated-screen design PDF (`PDF学習ワークスペース統合画面構成および基本仕様書.pdf`) and annotation data. They are historical reference material, not the current specification.
- `講義サンプル/第03回_Office変換/`: two Office-to-PDF validation results produced by the local LibreOffice runtime: one from PowerPoint and one from Word.

All PDFs, Office documents, and UI mockups are synthetic samples created within this project. They contain no externally sourced material or personal information.
The bundled release sample PDFs are fixed sample documents and will not be routinely replaced with future conversion quality updates.
